# CimBios.MetaModelTypeLib

**CimBios.MetaModelTypeLib** is a ready-to-use canonical model, which represents meta entities (Class, Property, Enum, e.t.c) like CimBios model objects. It's uses to design custom CIM profiles and type libraries for [CimBios library core](https://github.com/Cimbios/CimBios.Core).
